# Custom RC Crawler Telemetry System

Custom-built 1/10 scale RC crawler featuring a welded steel chassis and an ESP32-based telemetry system for real-time vehicle monitoring.

## Chassis & Structure

- Custom welded steel chassis
- 5/16 in steel tubing
- 3/16 in steel tubing
- Pothyes CNC aluminum center frame brace / skid plate / transmission mount

## Drivetrain

- INJORA complete SCX10 II gearbox transmission
- 13T pinion gear
- Alloy front steering axle with steel gears (SCX10 II)
- Alloy rear axle with steel gears (SCX10 II)
- MEUS Racing SCX10 II 313 mm link set
- HopsupRC steel driveshaft set (85–122 mm and 95–137 mm)
- HOPLEX heavy-duty driveshaft (125–185 mm)

## Suspension & Steering

- INJORA DRX 100 mm dual-rate oil shocks (4)
- LJWRC 30 kg waterproof digital steering servo

## Wheels & Tires

- 1.9 in aluminum beadlock wheels
- 1.9 in soft rubber crawler tires

## Electronics

- HobbyFans S3650 4300KV brushless motor
- HobbyFans 60A waterproof brushless ESC
- ESC programming card
- DumboRC X4 transmitter
- DumboRC X6F receiver
- OVONIC 2S 7.4V 5200 mAh 50C LiPo battery (Deans connector)

## Planned Telemetry System

- ESP32 microcontroller
- Hall-effect RPM sensor
- Battery voltage monitoring
- Motor temperature sensor
- Bluetooth mobile app interface

## Hardware

- 25 mm mounting screws
- Assorted metric screws and lock nuts
- Servo horn and steering linkage hardware
- Shock mounting hardware
