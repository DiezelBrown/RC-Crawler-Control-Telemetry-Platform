# Bill of Materials (BOM)

This document lists the major components used to build the Custom RC Crawler Telemetry System.

# Chassis & Structure

| Component | Description | Status |
|-----------|-------------|--------|
| Steel Chassis | Custom welded steel chassis | Installed |
| Steel Tubing | 5/16 in steel tubing | Installed |
| Steel Tubing | 3/16 in steel tubing | Installed |
| Center Frame Brace | Pothyes CNC aluminum center frame brace / skid plate / transmission mount | Installed |
